# Convex Backend

Run `pnpm convex` to start the dev server and generate `_generated/` types.

## Environment Variables (set via Convex dashboard)

- `CLERK_JWT_ISSUER_DOMAIN` — Your Clerk Frontend API URL (e.g. `https://verb-noun-00.clerk.accounts.dev`)
- `RESEND_API_KEY` — Resend API key
- `RESEND_FROM_EMAIL` — Verified sender address (e.g. `Axpo <noreply@yourdomain.com>`)
